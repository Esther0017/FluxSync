## 项目简介

**FluxSync** 是一个为中小型团队打造的实时协同编辑平台，支持多人同时在线撰写笔记、共享文档、自动保存并同步至云端。

核心特性：

- 多人实时协同编辑（基于 WebSocket + CRDT）
- 独立云存储服务，支持文件上传、历史版本、预览链接
- 文档组织与项目管理
- 细粒度权限控制
- 版本快照与文档历史恢复

适用于远程团队会议纪要、项目文档管理、头脑风暴笔记等场景。
